# Tomas Santos — Portfolio

Personal portfolio website for **Tomas Santos**, a Frontend Developer & UI/UX Lead specialising in React, animations, and high-performance web applications.

## About

Currently leading frontend development at **Bhout**, building dashboards serving tens of thousands of users across 8 club locations, real-time leaderboard systems used at events like **VivaTech**, **Web Summit**, and the **Jake Paul vs. Mike Tyson** event, and complex tools including CalDAV calendar integrations and dynamic team builders.

## Tech Stack

- HTML, CSS, JavaScript
- [GSAP](https://greensock.com/gsap/) for animations and scroll-driven interactions
- Google Fonts (Inter, Space Grotesk)
- Dark/light theme toggle
- Fully responsive design

## Running Locally

Simply open `index.html` in your browser — no build step required.

## Deployment

Deployed via [GitHub Pages](https://tomasgsantos.github.io/portfolio/) from the `main` branch.
